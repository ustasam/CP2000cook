




-- Встроенный язык (скрипт) --

    Комментарии - незначащие для скрипта пояснения к программе

    # Комментарий
    # Все, что после данного символа игнорируется скриптом
    // Комментарий
    // Все, что после данного символа игнорируется скриптом
    /* Комментарий */
    /* Все, что заключено в данные символы игнорируется скриптом */



    Параметры заключенные в квадратные скобки [] являются необязательными для команды.

    Идентификатор - имя параметра
    Значение - Число или идентификатор

    Вычисление значения идентификатора выполняется по формуле:
    Идентификатор = Идентификатор + Идентификатор

    Время - Целое число или заданная по формату Число:Число:Число значение
    обозначающее время.

    Допутстимые действия + - / *


    Команды скрипта:

    Имя  "Строка в кавычках"
    аналоги: Name или Заголовок

    Конец
    аналог: End

    Остановить
    аналог: Stop

    Сообщение  "Строка в кавычках"
    аналог: Message

    ОкноСообщения "Строка в кавычках"
    аналог: MessageDialog

    ОкноВвода Идентификатор [Значение или "Строка в кавычках"] ["Строка в кавычках"]
    аналог: InputDialog

    Ожидание [Время] ["Строка в кавычках"]
    аналоги: Wait Пауза Pause  Подтверждение Confirmation

    Гудок () [Целое] [Целое] [Целое] [Целое]
    аналоги: Beep Звук Sound

    Операция Значение "Строка в кавычках" Время [Время]
    аналог: Operate

    Повторить Значение code_block
    аналог: Repeat

    Отчет |") "Строка в кавычках" "Строка в кавычках" [Значение | "Строка в кавычках"]
    аналог: Report
